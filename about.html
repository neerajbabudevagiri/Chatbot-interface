About SmartChatBot

SmartChatBot is a lightweight, responsive chatbot interface created for educational and prototyping purposes. This project is designed to demonstrate how conversational interfaces can be built using HTML, CSS, and JavaScript.


It currently functions as an echo bot, where the user's message is returned as the chatbot's response. However, the system is structured to support more advanced features, including:

  • ✅ Multiple pages (Home, Chat, About, Contact)
  • ✅ Clean UI/UX with responsive layout
  • 🔄 Ready for backend integration (Node.js, Flask, etc.)
  • 🧠 Can be upgraded to use AI APIs (OpenAI, Rasa, etc.)
  • 🗃️ Extendable for persistent chat history, user profiles

Built with ❤️ by a passionate developer interested in AI and full-stack development.
